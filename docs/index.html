Many LPA planning portals do not provide a 'download all' button. This is that button.

Run PlanGrab on this computer

Downloading runs on your own machine, so files come from your IP (never a shared server) and save straight to a folder you choose. You just need to start the small PlanGrab helper — once.

  1. ⬇️ Download PlanGrab for Windows (~19 MB)
  2. Unzip it anywhere and double-click Run — no installation, no admin rights.
  3. If your browser asks to allow access to your local network, choose Allow.
  4. This page connects automatically once the helper is running.

Already running it, or not on Windows? This page connects as soon as the helper is up.

Documents are listed newest to oldest.
